# Задача 3 - LogFormatter Console App

Консольная программа и проект с тестами xUnit.

## Сборка и тестирование 

### Visual Studio: 

**Test Explorer** -> **Run All Tests**

### Консоль:

```bash
dotnet test
```

## Запуск программы из папки сборки

```bash
log-formatter
```

При необходимости, настройка параметров (изменение исходной директории с файлами, директории для нормализованных файлов и наименование файла с ошибками) производится через указание альтернативных значений в файле **appsettings.json** в папке сборки или в корневой папке основного проекта перед сборкой программы.
